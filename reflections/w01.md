# Lecture 1

This week, I decided to discuss the Agile v2 principle: Simplicity: The art of maximizing the amount of work not done.As there is no assigned project in week one yet, I tried applying this to preparing my local environment. Every single time I learn something new about programming languages, I tend to complicate my life with all sorts of unnecessary tasks. I spend endless hours trying to install useless VS code plugins, tweaking the terminal and creating complex folder structures, despite not having any task yet. So this week, considering that principle, I had the chance to experiment with how many unnecessary tasks I can skip altogether. I focused exclusively on the necessary tools I needed to start. I just installed git, ensured that I was using the correct Node version, and cloned the starter project.
